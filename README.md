# habit-wish-app

> 每一个程序员这辈子都会写一个 Todo App，我也不例外。
>
> 但我写完才发现——我不光有拖延症，我还不会坚持写 Todo。
>
> 所以这个 App 其实是写给我自己用的，顺便分享给同样需要被拯救的你。

---

## 🔥 这是什么

一个半正经的习惯养成 + 待办管理 App，帮你：

- 记录今天要做什么（然后明天继续愧疚）
- 养成好习惯（然后某天破功）
- 积攒金币（然后全部拿去兑换奶茶）
- 在星愿池许愿（然后发现愿望实现要 8848 枚金币）

---

## ✨ 功能一览

| 模块 | 能干啥 | 能不能坚持 |
|------|--------|------------|
| **待办** | 看板式任务管理，支持优先级 | 看心情 |
| **习惯** | 每日打卡，连胜机制，分类管理 | 看毅力 |
| **时间线** | 日历选择，查看每日安排 | 也许 |
| **星愿池** | 攒金币兑换奖励，长期目标 | 看金币够不够 |
| **设置** | 换头像、改昵称、开深色模式 | 一定可以 |

## 🛠️ 技术栈

```
框架:   uni-app (Vue 3)
状态管理: Pinia
UI组件:  uni-ui
存储:   localStorage (Sync版，异步是什么，能吃吗)
```

是的，这是一个用 `uni.getStorageSync` 同步读写localStorage的项目。

不是不会用异步，是因为——

> **同步代码写起来真香。**

## 📁 项目结构

```
habit-wish-app/
├── components/          # 组件，我写得很认真，虽然你不一定会看
├── pages/               # 页面，五个tab，不多不少刚刚好
├── store/modules/       # 状态管理，分了四个模块，比我衣柜还整齐
├── utils/               # 工具函数，日期计算和存储封装
└── static/              # 静态资源，图标是emoji，性能靠信仰
```
